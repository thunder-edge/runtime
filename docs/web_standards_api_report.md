## Node API Compatibility

This section is validated by runtime checks and is separate from Web Standards scoring.
Official compatibility levels for `node:*` APIs:
- `Full`: implementation considered functionally complete for the tested contract.
- `Partial`: implementation works for common/runtime-safe paths with documented limitations.
- `Stub`: module is importable, but unsupported methods fail deterministically when called.
Stub failures use the standardized format: `[thunder] <api> is not implemented in this runtime profile`.
Unsupported privileged behavior fails with deterministic errors (for example `EOPNOTSUPP`) instead of panicking or exposing host resources.
